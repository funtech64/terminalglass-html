Current paths

Three supported deployment paths

Customers choose a supported deployment location. Terminal.Glass remains the product they buy and use. We remove the CPU, memory, storage, GPU, operating-system, and cloud-instance guesswork.

NoCloudGPT — Ubuntu Linux VMs and standalone servers

Run Terminal.Glass on an Ubuntu Linux 24/26 virtual machine or standalone server you control — customer-owned, customer-controlled private AI on local, virtualized, or bare-metal infrastructure.

  • Ubuntu Linux 24/26 VM or standalone server

  • One installation script, one Terminal.Glass instance

AWS — customer-owned account
