Infrastructure

Hosting and Infrastructure Costs

The Terminal.Glass fee covers your private AI server deployment, configuration, documentation, and support. It does not include cloud servers, dedicated hardware, GPU instances, or third-party model usage.

Your private AI system may run on-premises, in a private datacenter, or on customer-owned cloud infrastructure. Managed hosting is available where offered — ownership and billing are confirmed before launch.

Final model capability depends on the server resources you choose. Some deployments use cloud-accelerated model inference with a small private Linux host. Those workflows are not fully offline.
