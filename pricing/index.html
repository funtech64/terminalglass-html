License rule

Glass Licenses are interchangeable. Deployment Keys are not.

A Glass License is the portable business capacity customers buy. A Deployment Key is the revocable credential for one active deployment or approved install.

What a Glass License can support

  • NoCloudGPT local/private host
  • YourCloudGPT customer-owned cloud host
  • Glass Agent capacity
  • Jet Agent model lane access inside the package
  • Future OpenClaw-Glass beta agent capacity

Additional licenses

  • Sunrise Starter additional Glass Licenses: $100 each
  • Starter customers reaching $400 total spend can receive a bulk gift upgrade into Group Plan structure
  • Group Plan additional licenses start at $50 each inside the same organization
  • Each added Group license drops by $5 until reaching a $25 floor
