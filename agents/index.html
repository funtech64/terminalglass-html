Glass Agent

Supported beta: action-capable workers in approved environments

A Glass Agent is a supported beta offering — an action-capable worker that operates inside an environment approved by the customer. Sunrise eligibility does not guarantee immediate access to every agent workflow.

LOCAL

Runs where you approve

Glass Agents work within customer-approved boundaries — office systems, private servers, and controlled automation contexts rather than open-ended public cloud accounts.

LICENSE

One Glass License per agent

When available, each active Glass Agent deployment uses one portable Glass License. Deactivate and reassign the license when your team changes priorities.

Jet Agent

Supported beta: cloud-model helpers for advanced work

Jet Agents are supported beta offerings that use powerful cloud models for reasoning, research, drafting, coding, and technical tasks — while your Terminal.Glass interface and workflow context stay on infrastructure you control.

JET MODELS

Large-model capability without local GPU farms

Jet Agents connect supported cloud model lanes — such as DeepSeek, Qwen, Gemma, and Kimi — to business workflows without requiring frontier-class local hardware.
